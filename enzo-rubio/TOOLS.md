# TOOLS.md - Enzo Rubio

## APIs & Data Sources

### Player Data
- **API-Football**: 1,200+ leagues, player stats, market value, contracts
- **Transfermarkt**: Player profiles, salary data, transfer history
- **WhoScored** (optional): Advanced statistics, performance data

### Media Monitoring
- **Mundo Deportivo**: Spanish football news
- **AS**: Spanish sports outlet
- **ESPN**: Global sports news
- **Marca**: Spanish sports
- **Goal.com**: International football
- **UOL** (Brazil): Brazilian sports news
- **Globoesporte** (Brazil): Brazilian football
- **ESPN Brasil**: Brazilian sports
- **Olé** (Argentina): Argentine football
- **La Nación** (Argentina): Argentine news
- **Fox Sports**: Latin American sports

### Core Tools (OpenClaw)
- **web_search**: Find player news, transfer rumors
- **web_fetch**: Extract player data, contract info from sources
- **message**: Send player profiles, deal updates to clubs
- **exec**: API calls to player databases

## Integration Priorities
1. **Phase 1**: API-Football (player search + stats)
2. **Phase 2**: Media monitoring (web scraping or RSS)
3. **Phase 3**: Transfermarkt (salary data, contract terms)
4. **Phase 4**: Advanced stats (WhoScored or similar)
